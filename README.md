# IoT Based Camera Detection System

## Overview
The IoT Based Camera Detection System is a smart surveillance project designed to detect objects or human presence using a camera module and IoT technology. The system captures images or video, processes the data, and sends notifications to the user for real-time monitoring and improved security.

## Objectives
- Detect human or object movement.
- Provide real-time monitoring.
- Improve home and office security.
- Enable remote access through IoT.

## Components Used
- ESP32-CAM
- Camera Module
- Wi-Fi Network
- Power Supply
- Jumper Wires

## Software Used
- Arduino IDE
- Embedded C++
- IoT Platform

## Working Principle
The ESP32-CAM continuously monitors the surroundings using its camera. When motion or a predefined object is detected, the system captures an image and sends it to the user through an IoT platform or mobile application.

## Features
- Real-time image capture
- Wireless monitoring
- Motion detection
- Low-cost security solution
- Remote access

## Applications
- Home Security
- Office Surveillance
- Smart Buildings
- Industrial Monitoring
